# 服务条款 — X Article Markdown Paste

**最后更新：2026 年 5 月**

安装或使用 X Article Markdown Paste（以下简称"本扩展"）即表示同意以下条款。

## 本扩展是什么

一款 Chrome 浏览器扩展。功能：在 X (Twitter) 文章编辑器内拦截粘贴事件，把 Markdown 文本转换为 X 文章的原生格式（标题、列表、代码块、分隔线、推文嵌入、图片、表格等）。

本扩展**不是 X (Twitter) 官方产品**，与 X 公司无任何关联。

## 免费版与 Pro 版

**免费版：**
- 全部文本格式、代码块、分隔线、内联标记、链接
- 网络图片每次粘贴最多 3 张
- 推文嵌入每次最多 1 条
- 不支持表格渲染
- 文末附 `✨ via X-MD-Paste` 签名（可手动删除）

**Pro 版（通过 license key 激活）：**
- 解锁全部数量限制
- 解锁表格 → PNG 渲染
- 无文末签名
- 后续新增功能（AI 改写、本地图片支持等）优先体验

## 定价
