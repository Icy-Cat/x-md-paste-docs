## What the Extension is

A Chrome extension that intercepts paste events inside the X (Twitter) article editor and converts Markdown text into X's native rich-text format (headings, lists, code blocks, dividers, tweet embeds, images, tables).

The Extension is **not affiliated with X (Twitter)**.

## Free vs Pro

**Free tier:**
- All text formats, code blocks, dividers, inline marks, links
- Up to 3 network images per paste
- Up to 1 tweet embed per paste
- No table rendering
- Footer signature `✨ via X-MD-Paste` (manually deletable)

**Pro tier (license-key activated):**
- Unlimited everything
- Table → PNG rendering
- No footer signature
- Priority access to upcoming features (AI rewriting, local image support, etc.)
